# Sistema de Lanchonete (Java)

Projeto desenvolvido em Java, utilizando Programação Orientada a Objetos.  
O sistema permite cadastrar, listar e gerenciar produtos do cardápio, como hambúrgueres e bebidas.  
Também será integrada uma conexão com banco de dados para armazenamento persistente.

## Tecnologias
- Java
- POO
- ArrayList
- Estrutura modular por pacotes
- Integração futura com banco de dados (JDBC)
## Conceitos de POO Utilizados
- Encapsulamento (atributos privados com getters/setters)
- Herança (Hamburguer e Bebida herdando de Produto)
- Polimorfismo (uso de Produto como tipo genérico na lista)
- Classe abstrata (Produto)
- Sobrescrita de métodos (gerarDescricao() e calcularPrecoFinal())
## Funcionalidades
- Cadastro de hambúrgueres e bebidas
- Listagem completa do cardápio
- Busca de produtos por ID
- Remoção de produtos
- Cálculo de preços
- Estrutura pronta para integração com banco de dados
